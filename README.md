## üß± How the Project Works

### Step 1: Build the MySQL Database

The backend includes:

- **Tables** for products, orders, shipments, and inventory  
- **Views** for summaries (e.g., product history, stock reports)  
- **Stored Procedures** for operations (e.g., mark order as received, update stock)  
- **Functions** for business logic (e.g., check if a product needs restocking)

This simulates how real businesses organize and automate data workflows.

---

### Step 2: Build the Streamlit Frontend

The frontend built with Streamlit allows users to:

- View data from tables and views  
- Run stored procedures using buttons  
- Insert/update records through forms  
- Execute MySQL functions directly from the UI  
- View instant results‚Äîno SQL queries needed
